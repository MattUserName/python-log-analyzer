\# Python Log Analysis \& Suspicious Login Detector



\## Overview

This project simulates basic SOC alert detection logic by analysing authentication logs for repeated failed login attempts.



\## Features

\- Parses log files

\- Counts failed login attempts per IP address

\- Detects potential brute-force attacks using configurable threshold



\## Technologies Used

\- Python 3



\## How to Run

auth.log is a text file mimicking authentication network logs.
